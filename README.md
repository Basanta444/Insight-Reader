# Insight Reader

A digital reading and knowledge-management platform — search a categorized library, track reading progress in real time, and capture personal notes ("insights") linked to every book you read.

**Live Demo:** [insight-reader-khaki.vercel.app](https://insight-reader-khaki.vercel.app)
**Repo:** [github.com/Basanta444/Insight-Reader](https://github.com/Basanta444/Insight-Reader)

## Features

- Email/password & Google OAuth authentication
- Search and filter a 240+ book library by title, author, description, or category
- Real-time, page-level reading progress tracking, synced via Firestore
- **Insight Dashboard** — full CRUD for personal notes linked to specific books
- Star ratings & written reviews for community discovery
- Smart book recommendations based on category/author
- Bookmarking, dark/light mode, fully responsive UI

## Tech Stack

React (Vite) · React Router · Firebase (Auth + Firestore) · Vercel
